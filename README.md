# 🌌 NEBULA — 星云粒子交互体验

> 摄像头捕捉手势 · 全屏粒子实时响应 · 科幻疗愈数字艺术

**NEBULA** 是一款面向碎片化时间的解压交互艺术品。打开浏览器，允许摄像头，用双手与 7,000+ 发光粒子互动。无需学习，即刻沉浸。

## ✨ 特效系统

| 特效 | 描述 |
|---|---|
| 🌌 **星云背景** | 实时着色器渲染青紫星云，Simplex 3D 噪声 |
| 💫 **粒子拖尾** | 每个交互粒子带拖尾光流，形成能量河流 |
| ✋ **手部全息线框** | 21 关键点发光连线 + 品红关节节点（钢铁侠 HUD 风格） |
| ⭕ **脉冲能量环** | 手势切换时扩散冲击波光环 |
| ✨ **星座连接网** | 近距离粒子间动态细线，神经网络拓扑 |
| ⚡ **双手能量桥** | 双手间贝塞尔曲线等离子束 |

## 🎮 手势交互

| 手势 | 效果 | 情感 |
|---|---|---|
| ✋ 张开手掌 | 粒子吸引聚拢，轨道环绕 | 接纳 |
| ✊ 握拳 | 粒子爆炸飞散 | 释放 |
| ☝️ 食指向上 | 指尖火花喷射，向上光柱 | 创造 |
| ✌️ V字 | 粒子优雅分流 | 秩序 |
| 🤟 摇滚手势 | 粒子螺旋上升 | 自由 |
| 👍 拇指向上 | 粒子上浮变亮 | 积极 |
| 👎 拇指向下 | 粒子下沉沉淀 | 沉思 |

## 🎵 程序化音频

纯 Web Audio API 生成，零音频文件：
- **环境底音**: 3 失谐正弦波 + LFO，宇宙微波背景嗡鸣
- **粒子声音**: 60 音频粒子池，空间化星尘雨
- **手势钟声**: D 小调五声音阶，每种手势独特音符

## 📱 全设备适配

| 设备 | 粒子配置 | 特效 |
|---|---|---|
| 🖥️ 桌面 | 5,000+2,000 | 全开 |
| 📱 iPad | 4,000+1,500 | 全开 |

支持触摸控件（双击全屏、底部上滑提示）和键盘快捷键。

## ⌨️ 快捷键

| 按键 | 功能 |
|---|---|
| `F` | 全屏切换 |
| `H` | 显示/隐藏手势提示 |
| `M` | 静音切换 |

## 🚀 技术栈

- **Three.js** 0.182 — WebGL 渲染引擎
- **MediaPipe** GestureRecognizer — 手势识别 (8 种内置手势)
- **Web Audio API** — 程序化音频合成
- **单 HTML 文件** — 零构建，CDN 依赖，拖拽即用
